Speaking to the Web

Introduction

JavaScript Speech is my hands-on journey to mastering how to make web pages come alive.
Starting with the core language fundamentals, I progressed through DOM manipulation, interactivity, and real-world user experience enhancements.
This curriculum is guided by practical weekly challenges - often inspired by Milk and Eel - and filled with dynamic, often absurd, mini-projects that made the learning fun.

Topics Covered

  • Week 1: JavaScript Foundations

    Syntax, variables, logic operators, conditionals - plus a mood-based Milk response system.

  • Week 2: Functions, Loops, and Events

    Reusable logic, for/while loops, event listeners, and Eel-based button interactions.

  • Week 3: DOM Manipulation Basics

    querySelector, textContent, class toggles, and spider hover effects.

  • Week 4: JSON, Objects, and Lists

    Parsing and generating JSON, looping through data, and a dynamic quote viewer.

  • Week 5: Forms, Validation, and Feedback

    Manual input validation, error messaging, and a fully animated message loader experience.

Topics Left

  • Week 6: Classes, Modules, and Cleaner Code - Intro to JS classes, reusable module structure, and organizing your codebase with clarity.
  • Week 7: Local Storage & Persistence - Saving and loading user data using localStorage — from themes to input memory.
  • Week 8: Animations & Timers - Timers, animation loops, and blending CSS/JS into smooth interactive behavior.

Outcome

To be filled upon completion
By the end of this path, I aim to be confident writing clean, interactive JavaScript - able to manipulate any page, handle real-time user input, load and display structured data, and enhance accessibility and feedback through DOM control.
Plus: a deeply personal JS library of Milk and Eel absurdities.
